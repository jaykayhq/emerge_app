# Rive Asset Placeholder

This is a placeholder file for the archetype_athlete.riv animation.

To create the actual asset:
1. Open https://rive.app and create a new file
2. Design an athletic character with:
   - Strong, determined expression
   - Athletic outfit (tank top, shorts)
   - Power pose / flexing animation
3. Add a State Machine named "State Machine" with:
   - `isSelected` (Bool) input
   - `select` (Trigger) input
   - States: idle, selected, celebrate
4. Export as archetype_athlete.riv and replace this file
