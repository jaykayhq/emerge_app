# Emerge World Backgrounds — Image-to-Video Prompts

**Purpose:** Animate each of the 15 static PNGs into seamlessly looping video backgrounds.  
**Workflow:** Feed the corresponding PNG as the source image into your video generation tool, then use the prompt below.

---

## IMPORTANT — Shared Rules (apply to ALL video prompts)

- **Source:** Use the matching `.png` file as the input image
- **Duration:** 4–6 seconds, seamlessly looping (ping-pong or true loop)
- **Camera:** Completely stationary — **no pan, no tilt, no zoom, no camera movement whatsoever**
- **Motion style:** Subtle ambient animation only — this is a background, never the focus
- **Audio:** None
- **Format:** MP4, H.264, portrait 1080×2340
- **Loop:** Must loop invisibly — end frame must match start frame
- **No characters, no UI, no text should appear**

## 🌲 Living Forest

### Source: `forest/thriving.png` → `forest/thriving.mp4`
```
Animate this lush forest scene with subtle ambient life. God rays of golden sunlight slowly shift and pulse through the canopy. Individual leaves gently sway in a soft breeze. ```
```
