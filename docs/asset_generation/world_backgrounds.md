# Emerge World Background — MixBoard Generation Prompts

**Purpose:** Generate 15 portrait-orientation PNG images for the World Background system.  
Each image is a full-screen background (9:19.5 portrait). Generate at **1080×2340px minimum**.

---

## IMPORTANT — Shared Style Rules (apply to ALL prompts)

- **Orientation:** Tall portrait, fills a mobile screen from top to bottom
- **No text, no UI, no characters, no watermarks**
- **Dark at the bottom** — the lower 40% should be naturally darker so white UI text is readable
- **Slightly dark/vignette at the top** — the upper 15% should fade darker for top bar legibility
- **Cinematic quality** — wide dynamic range, atmospheric depth, painterly or photorealistic
- **Do not use HDR clipping** — no blown-out whites

## Video Upgrade (Future)

When generating video loops, use the same prompt but specify:
- **Duration:** 4–8 seconds seamlessly looping
- **Format:** MP4, H.264, no audio
- **File name:** Same path, `.mp4` extension (e.g. `forest/thriving.mp4`)

The widget will automatically detect and use `.mp4` files when present.
