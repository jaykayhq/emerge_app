Reimagining the Tribe Sanctum: Layout Options

Select the visual layout that best fits the Emerge Pro "Metropolis of Focus" HUD design system.

A

Approach A: Glassmorphic Sub-Tabs (Recommended)

Clean, tabbed HUD layout within the Tribes section. Users switch between Sanctum, Quests, Members, and Bonds using a glass-styled capsule selector directly under the App Bar. Dedicated views for each sub-feature keep scroll-depth short.

B

Approach B: Long-Scroll with Sticky Anchor HUD

Keeps all widgets on a single long scrolling page, but introduces a sticky sub-navigation anchor bar. Clicking an item smooth-scrolls the view down to the corresponding section (Quests, Leaderboards, Activity, etc.).

Approach C: Floating Hub Nodes

A game-like layout where the Sanctum screen displays only the emblem and core progress metrics. Floating glowing orbit nodes branch off the emblem. Tapping a node slides in a modal/overlay sheet with detailed view for Quests, Members, or Bonds.
