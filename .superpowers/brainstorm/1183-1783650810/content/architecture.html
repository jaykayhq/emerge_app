  • Set Reward: text field + suggestion chips. Saves on blur.

  • Delete Habit: confirmation dialog → calls habitRepository.deleteHabit.

3 · Data flow & new entities

  • Existing: DailyReflection = one-per-day global. Unchanged.
  • NEW: HabitReflection = one-per-(userId, habitId, localDate). Mirrors the existing daily reflection schema but adds habitId and scopes reads by habit.
  • NEW table: habit_reflections in drift, unique index on (userId, habitId, localDate). DAO mirrors DailyReflectionsDao.
  • NEW repo: HabitReflectionRepository with getForHabit(userId, habitId, date) + save(). Remote mirror: subcollection users/{uid}/habit_reflections/{autoId}.
  • NEW provider: habitReflectionProvider(userId, habitId, date) for sheet; saveHabitReflectionProvider for write.
  • Habit edits (priming, reward): already supported via habitRepository.updateHabit; sheet calls it directly.
