3D Stickman — Options & Demo

You want 3D. Here's the landscape for 3D stickmen in Flutter.

Procedural 3D (Canvas)

Pure Flutter, no deps

Use Flutter Canvas to project 3D points → 2D. Render bones as 3D cylinders. Rotate with matrix math. Like the 2D system but with depth.

  • Zero dependencies
  • 60fps guarantee
  • Same codebase
  • No built-in lighting
  • Manual depth sorting
  • No mesh loading

Three.dart / ThreeFlutter

Three.js port to Dart

OpenGL ES rendering. Full 3D pipeline: cameras, lights, materials, skeletal animation. Can load glTF/glb models.

  • Full 3D pipeline
  • glTF model support
  • Lighting/shadows
  • ~5MB package size
  • Complex build setup
  • Platform quirks (web vs native)

Rive 3D

Designer-friendly

Build in Rive editor, export as .riv. Bone-based animation with mesh deformation. Excellent for characters.

  • Visual editor
  • State machine
  • Cross-platform
  • Runtime cost
  • Learning curve for editor
  • Customization at runtime is harder

Custom GL (shaders)

Maximum control

Write vertex/fragment shaders for stickman rendering on GPU. Full control over bone matrices, skinning, and custom effects.

  • Maximum performance
  • Custom shader effects
  • GPU-accelerated skinning
  • Steep learning curve
  • Shader debugging is hard
  • GLSL per platform
