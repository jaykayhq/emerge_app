Soft 3D Rounded Figure — Technical Approaches

Three ways to get soft 3D rounded characters in Flutter, matching the reference image style.

🎨

A — Procedural 3D

Pure CustomPainter — draw capsules, spheres, and ellipses with radial gradients to fake 3D lighting. Think "paper doll" meets 3D.

Pros:
  • Zero dependencies
  • 60fps guaranteed
  • Full runtime color/part swap
  • Looks like image at display size
Cons:
  • Not true 3D — fixed angle
  • No model rotation
  • Manual shading code
🧊

B — ThreeFlutter

Full 3D engine in Flutter — render spheres/cylinders with real lights, shadows, subsurface scattering. Rotatable, zoomable.

Pros:
  • True 3D — all angles
  • Real lighting/shadows
  • Identical to reference image
  • Animated rotation
Cons:
  • ~5MB package
  • Platform-specific quirks
  • Complex build setup
🎭

C — Rive

Design in Rive editor — bone-based mesh deformation with vector fills that look 3D. Export .riv, animate with state machine.

Pros:
  • Visual editor
  • Bone animation built-in
  • Small runtime
  • Cross-platform
Cons:
  • Limited 3D (2.5D mesh)
  • Runtime customization hard
  • Must use Rive editor

My Recommendation

⭐

Approach A — Procedural 3D (Recommended)

Why: For an avatar system that needs runtime customization (swap colors, add equipment, show evolution), procedural rendering is the most flexible. You can change any part's color/shape at runtime without re-exporting models. It performs at 60fps, has zero dependencies, and at avatar display size (~80-150px) the soft gradient shading looks indistinguishable from true 3D.

ThreeFlutter (B) is overkill for an avatar that's typically displayed at 100px — you'd fight platform quirks for minimal visual gain at that size. Rive (C) is great for pre-baked animations but makes runtime equipment/color swapping painful.

Compromise: If you want true 3D rotation (spin the avatar on the profile screen), we can embed a small ThreeFlutter view just for the main avatar display, while the procedural renderer handles in-game/in-list thumbnails and customization previews.
